    Project Objective#

    The end goal of this project is to build better society by helping certain individuals to become better and best, by providing concise action guides and also more detailed info for those who have keen interest. Action guides designed in a way, that a reader, with a least amount of attention can start implementing them immediately. It is recommended, to follow the order in which guides appear.
